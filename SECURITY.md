# Security Policy

## 🔒 Supported Versions

| Version | Supported          |
| ------- | ----------------- |
| 2.0.x   | ✅ Active        |
| 1.x     | ⚠️ End of Life   |

## 🚨 Reporting Vulnerabilities

If you find a security vulnerability:

1. **DO NOT** create a public issue
2. Email directly: (contact maintainer)
3. Include details:
   - Type of vulnerability
   - Full paths of affected files
   - Steps to reproduce
   - Potential impact

## ✅ What We Promise
- Acknowledge reports within 48 hours
- Provide timeline for fixes
- Credit reporters (with permission)

## ⚠️ Scope
This tool is for:
- Educational purposes
- Testing accounts you own
- Security research with permission

Not for:
- Unauthorized access
- Data theft
- Malicious activities
