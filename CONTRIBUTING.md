# Contributing to Nebula Checker

## 🤝 Welcome Contributors!

Thanks for your interest in Nebula Checker. This project welcomes contributions from the community.

## 📋 Guidelines

### What You Can Contribute
- ✅ Bug fixes
- ✅ UI improvements  
- ✅ New features
- ✅ Documentation
- ✅ Translations

### Development Setup
```bash
# Clone the repo
git clone https://github.com/tajcactus23-rgb/Nebula-Checker-.git
cd Nebula-Checker-

# Create a feature branch
git checkout -b feature/your-feature

# Make your changes
# Test locally with: python -m http.server 8080

# Commit and push
git commit -m "Your feature description"
git push origin feature/your-feature
```

### Pull Request Process
1. Fork the repository
2. Create your feature branch
3. Test thoroughly
4. Submit a PR with clear description

## 📝 Code Style
- Keep it clean and readable
- Comment complex logic
- Use consistent formatting

## ⚠️ Restrictions
- No malicious features
- No external data collection
- Respect the license

**Questions?** Open an issue for discussion.
